# Password-Manager

Password Managing Python CLI tool providing sign in/up and additional functions.

## Feature:
- Saving Username/Password/FavoriteFood in a given JSON file
- Sign Up
- Sign In
- Change Password
- Remove Account
- Main Page

## Skills Practiced:
- Parameterization
- File I/O
- Exception Handling
- OOP
- Refactoring
- Serialization

## Takeaways:
- Removing legacy is the first step of adding a new one.
- Running code tells that reading doesn't.
-  A bug's locations might not be indicated in the error line.
